This document describes the architecture and the API of the Othello which is the result of the second assignment work of the SS/14 for the subject programming II.

Task description of the assigment

It's to develop a application which simulates the game othello for two players and meet the following tasks:

  • player vs player
  • displays the stones of each player
  • the game end if all stone are set
  • custom game field size[6-10]
  • graphical user interface

The Reversi Application

The application is split into three parts:

  1. The game logic objects [org.yvka.Beleg2.game.*] - which contains the classes of the game logic.

  2. The console user interface [org.yvka.Beleg2.console.*] - which contains only classes which implements the console user interface.

  3. The gui user interface [org.yvka.Beleg2.gui.*] - which contains only classes which implements the graphical user interface.
